# 🏫 캠퍼스 맵 프로젝트 백엔드 (Campus Map Backend)

본 레포지토리는 캠퍼스 건물 정보 및 학과 위치, 검색 기능을 제공하는 백엔드 서버입니다.

## ⚙️ 시작하기 전 준비사항
* **Node.js**가 설치되어 있어야 합니다. (버전 18 이상 권장)

## 🚀 로컬 서버 실행 방법
1. 모듈 설치: 터미널에 아래 명령어를 입력하여 필요한 라이브러리(`express`, `cors`)를 설치합니다.
   \`\`\`bash
   npm install
   \`\`\`
2. 서버 실행:
   \`\`\`bash
   node server.js
   \`\`\`
3. 서버가 정상적으로 켜지면 터미널에 `http://localhost:3000 에서 실행 중입니다.` 메시지가 표시됩니다.

---

## 📡 API 명세서 (API Reference)
프론트엔드에서는 아래 주소로 `fetch` 또는 `axios` 요청을 보내어 데이터를 사용할 수 있습니다. (CORS 에러 해결 완료)

### 1. 전체 건물 목록 조회
* **URL:** `GET http://localhost:3000/api/buildings`
* **설명:** 캠퍼스 내의 모든 건물 데이터를 배열 형태로 반환합니다.

### 2. 특정 건물 상세 조회
* **URL:** `GET http://localhost:3000/api/buildings/:id`
* **예시:** `http://localhost:3000/api/buildings/A01`
* **설명:** 건물 고유 ID를 기반으로 단일 건물 객체의 상세 정보를 반환합니다.

### 3. 🌟 통합 검색 엔진 (추천)
* **URL:** `GET http://localhost:3000/api/search?q={검색어}`
* **예시:** `http://localhost:3000/api/search?q=정보`
* **설명:** 건물 이름, 영문명, 약칭뿐만 아니라 **건물 내에 소속된 학과 이름**으로도 건물을 찾아냅니다. 
* **프론트엔드 참고사항:** 검색창 컴포넌트를 만들 때 이 API 하나만 호출하면 모든 필터링 로직이 서버에서 처리되어 결과가 반환됩니다!

## 📂 주요 파일 설명
* `server.js`: 메인 라우팅 및 검색 로직이 포함된 서버 코드
* `buildings.json`: 캠퍼스 건물, 좌표, 학과, 편의시설 정보가 담긴 메인 데이터베이스
